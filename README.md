
# IP Information Browser

An IP Information Browser


## Installation

Install and run locally info-ip-browser with npm (clone the project before)

```bash
  cd info-ip-browser
  npm install
  npm run dev
```
    
## Tech Stack

**Client:** React, PicoCSS

**Used API:** https://ipgeolocation.io/
